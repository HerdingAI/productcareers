# `open-source-job-board`

An open source job board built with Supabase.


## Built with

- Backend: PostgreSQL + Supabase
- Frontend: Next.js + Tailwind
- Auth: Supabase Auth + Postgres Row Level Security
- Storage: Supabase Storage
